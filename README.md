# Project 13 Foundation

可直接部署到 GitHub Pages 的靜態 PWA，不需要安裝或建置。

## 已完成功能

- 分層首頁：背景、HUD、旅伴、13 角色卡、貓咪、漂流瓶、世界留影與底部功能列互相獨立。
- 世界紀錄、今日旅程、廚房／身體紀錄、回憶館、Boss 圖鑑。
- 現實照片與世界留影可分開篩選、保留／取消保留、永久刪除。
- 動態 EXP、貝殼、星星、漂流瓶系統，並保留資源來源紀錄。
- 手機與電腦響應式排版、PWA 安裝與離線快取。
- 原創循環 BGM〈窗邊旅程〉，包含播放／暫停與音量控制。

## GitHub Pages

把本資料夾內所有檔案上傳到 Repository 最外層，Pages 設定選 `main` 與 `/(root)`。

所有生活資料只會儲存在目前裝置瀏覽器的 Local Storage 與 IndexedDB。清除瀏覽器網站資料會一併刪除紀錄。
